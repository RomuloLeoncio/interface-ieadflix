 # IEADFLIX:play_or_pause_button:

O presente projeto tem o objetivo de criar uma interface de página para uma igreja sob os padrões utilizados pela NETFLIX! Com esse projeto podemos:

- Criar uma interface agradável para os membros de igrejas poderem assistir cultos **ao vivo** , seminários, pregações, e outras atividades que tenham perdido ao longo da semana;
- Desenvolver uma nova plataforma digital de mídia como filmes, desenhos e músicas;
- Acesso ao design de carrossel para fácil escolha do usuário no conteúdo pretendido;



**Desejo que você aproveite esse modelo!**
